# Kronos Signal

Kronos Signal is a program add-on for **Kodi 21.x (Omega)** designed for **LibreELEC** systems. 
It provides a lightweight status overlay to keep users informed about key system conditions while maintaining stability and performance.

---

## Features
- **HUD Overlay** – Displays system information briefly on startup. 
- **CPU & RAM Monitoring** – Shows usage values for quick diagnostics. 
- **VPN Guard** – Confirms VPN initialization and status at boot. 
- **Abort-Safe Timing** – Uses responsive delays to avoid lockups during shutdown. 
- **Lightweight Execution** – Minimal runtime footprint, designed for seamless integration.

---

## Compatibility
- Kodi **21 Omega** 
- LibreELEC 12.0.2+ 
- Python 3 (bundled with Kodi)
